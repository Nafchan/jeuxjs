			PRE REQUIS DE L'APPLICATION	

******************************************************************************************************

POUR LANCER L'APPLICATION:
-------------------------- 

-Connexion Internet exigé!
				
-Installer nodeJs
				
-Ouvrir le terminal, vérifier que npm a été installé avec la commande "npm -v"
				
-Ensuite taper "npm install -g angular-cli"
				
-Copier le dossier du projet où vous souhaitez dans votre ordinateur
				
-Ouvrir un terminal de commande, puis se positionner dans le dossier ou se trouve les fichiers du projet
				
-Tapez "npm install"
				
-Quand l'installation du npm a été effectué, tapez "npm start"pour lancer le serveur du projet est lancé
				
-Allez sur le navigateur de votre choix puis tapez "localhost:4200"
				
-L'application s'affiche : Bonne navigation 

******************************************************************************************************

				ELEMENTS DU DOSSIER
